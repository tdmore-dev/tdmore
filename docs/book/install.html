TDMore: an R package for therapeutic drug monitoring

Chapter 2 Installation

TDMore depends on package RxODE, which requires a working C and fortran compiler to work. Installation procedure can be found here. Once RxODE installed, simply execute the following command in the R console:

devtools::install_github("tdmore-dev/tdmore")

If you would like to install the package with all its vignettes, please run the following command instead:

devtools::install_github("tdmore-dev/tdmore", build_vignettes = TRUE)
